Curso de Desenvolvedor Front-End

Senai Maracanã

O objetivo do curso é conhecer os pilares de uma aplicação web,
identificar a diferença entre HTML, CSS e JavaScript
assim como saber escrever um documento HTML com suas principais tags,
fazer um arquivo CSS com suas propriedades básicas: margin, width.
height, padding, color, background, border e
outros e saber utilizar os conceitos básicos de lógica de programação em JavaScript
(variáveis, condicionais, loops, entre outros)

    Unidades Curriculares do Curso

  • Iniciação a Tecnologia da Informação
  • Lógica de Programação
  • Front-End Essencial
  • Desenvolvimento Web
  • Integração de Competências Profissionais
